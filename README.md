# 🎨 Конструктор обложек для ВКонтакте

Универсальный инструмент для быстрого создания обложек сообщества ВК.  
Позволяет настраивать текст, фон, логотип, позиции элементов, шрифты и стиль кнопки.  
Два формата: главная обложка (1920×768) с визуализацией безопасной зоны для мобильных и живая обложка (1080×1920).  
Экспорт в PNG одним нажатием.

## ✨ Возможности

- **Настройка текста:** заголовок, подзаголовок, текст кнопки.
- **Градиентный фон:** выбор двух цветов.
- **Логотип:** загрузка своего изображения, регулировка размера и положения (X, Y).
- **Шрифты:** выбор из 4 вариантов (Inter, Montserrat, Roboto, Georgia).
- **Позиционирование:** слайдеры для точной настройки вертикального расположения заголовка, подзаголовка и кнопки.
- **Стиль кнопки:** белая, прозрачная с обводкой, зелёная в тон фона.
- **Пресеты:** быстрая смена темы (бизнес, образование, еда, путешествия, технологии, эко/сад).
- **Визуализация безопасной зоны:** жёлтая рамка на главной обложке показывает область, которая гарантированно видна на мобильных устройствах.
- **Экспорт в PNG:** отдельные кнопки для главной и живой обложки.

## 🚀 Как пользоваться

1. Откройте [страницу инструмента](https://merdigan.github.io/social-cover-studio/) (после включения GitHub Pages).
2. Настройте текст, фон, логотип и другие параметры в левой панели.
3. Нажмите «Обновить превью» (или просто измените любой параметр, превью обновится автоматически).
4. Проверьте, что важные элементы попадают в жёлтую безопасную зону.
5. Нажмите «Скачать PNG» для нужного формата.
6. Загрузите полученные изображения в сообщество ВК в качестве обложки.
